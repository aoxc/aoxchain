# Quantum Near-Closure Evidence Package

A candidate claiming quantum-full near-closure must include all evidence classes below.

## Required Evidence

1. deterministic simulation matrix covering mixed profile peers;
2. downgrade rejection proof bundle;
3. migration drill logs with artifact hashes;
4. rollback rehearsal records;
5. operator runbook validation outputs;
6. updated residual risk statement.

## Traceability Requirements

Each evidence item must include:

- candidate commit SHA,
- command transcript or deterministic runner identifier,
- artifact path/hash,
- generation timestamp,
- responsible operator/reviewer identifier.

## Reproducibility Rule

Any evidence item that cannot be reproduced under the documented command path is non-authoritative and fails near-closure evaluation.
