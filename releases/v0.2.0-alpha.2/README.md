# AOXC Release v0.2.0-alpha.2

This directory is generated by `scripts/release/prepare_repo_release.py` and contains version-scoped, operator-facing release artifacts.

## Artifact Scope

- target label: `linux-amd64`
- binaries in scope:
- `aoxc-linux-amd64`
- `aoxchub-linux-amd64`
- `aoxckit-linux-amd64`

## Directory Map

- `manifest.json`: machine-readable release index.
- `checksums.sha256`: SHA-256 for every published binary artifact.
- `compatibility.toml`: compatibility contract (network/profile/schema).
- `sbom.spdx.json`: SPDX SBOM surface for supply-chain review.
- `provenance.intoto.jsonl`: in-toto statement log placeholder.
- `certificates/`: release certificate metadata and certificate notes.
- `binaries/`: target-scoped binary payloads.
- `signatures/`: detached signature output location.
