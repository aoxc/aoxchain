# AOXCHub Command Catalog

The catalog is immutable at compile time and rendered as both operator-facing labels and technical previews.

## AOXC Diagnostics
- `aoxc doctor`
- `aoxc version`
- `aoxc build-manifest`
- `aoxc node-connection-policy`

## Make Workspace and Packaging
- `make build`
- `make test`
- `make check`
- `make package-all-bin`

## Network Placeholders and Operational Extensions
- `make net-mainnet-start`
- `make net-mainnet-status`
- `make net-testnet-start`
- `make net-testnet-status`

## Policy Notes
- Command previews are read-only and generated from immutable command specs.
- MAINNET and TESTNET availability can differ for specific actions.
